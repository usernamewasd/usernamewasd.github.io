额外加成（均为简单加法 / 最终倍率）
• 基伤加成：直接加在总基伤区（例：原基伤2.0 + 加成0.6 = 2.6）
• 元素加成：直接加在总元素区（例：元素3.4 + 加成0.6 = 4.0）
• 暴击加成：直接加在暴击率 x 上（例：x=2.6 + 加成2.0 = 4.6）
• 爆伤加成：直接加在爆伤倍率 y 上（例：y=4.5 + 加成0.5 = 5.0）
• 额外伤害加成：最终伤害 × (1 + 此值)，例：1 + 0.3 = 1.3 倍

⚡ 暴击等级处理模式（整体控制）
• 期望伤害：x=x，使用暴击期望公式 f = 1 + x·(y-1)，计算平均伤害
• 伤害下限：x=⌊x⌋（向下取整），按最低暴击等级计算，保守估计
• 伤害上限：x=⌈x⌉（向上取整），按最高暴击等级计算，乐观估计
• 自定义阈值：小数部分≥阈值则向上取整，否则向下取整
  例：阈值0.8时，x=4.7→等级4，x=4.8→等级5，x=5.0→等级5
• 暴击等级N的伤害倍率 = 1 + N·(y-1)
• 不同模式会影响伤害值，进而可能改变配卡排名

缩放规则：圣洁执法者时紫卡全体 ×125/135 | 所有额外加成为简单加法 | 额外伤害加成为最终乘算 | 暴击等级处理影响伤害计算
